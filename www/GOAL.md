# Image signing website
In the www folder, create a simple website that allows signing of image files (TIF, PNG, JPG).
It should have a menu with options "Sign", "Check", "Update".
It should use the library in src to sign, check, and update signatures of image file.
Use the logo in photo-verify-logo-transparent.png on the website.
Make it a single page website.

## Sign
A file must be uploaded (TIF, PNG, JPG).
The file is signed and downloaded to the user.

## Check
A file must be uploaded (TIF, PNG, JPG).
The signature check if it exists and the chain of custory displayed.
If no signature exists, the website says that the file is not signed.

## Update
A file must be uploaded (TIF, PNG, JPG).
The file is signed and downloaded to the user.

Create a CLAUDE.md in www to describe the site functionality.
